Installation

How Install MongoDB?(Windows)

  • mongodb.org/downloads
  • https://docs.mongodb.org/manual/tutorial/install-mongodb-on-windows/
  • Execute Installer
  • Create Default DB Path - "C:/Data/db"


How Install MongoDB?(Mac)

Using Homebrew

  • Update Homebrew’s package database brew update
  • Install MongoDB brew install mongodb
  • Create Default DB Path - "/data/db"


How Install MongoDB?(Manually/Linux)

  • mongodb.org/downloads
  • https://docs.mongodb.org/manual/tutorial/install-mongodb-on-os-x/
  • Extract the files from the downloaded archive tar -zxvf mongodb-osx-x86_64-3.0.4.tgz
  • Copy the extracted archive to the target directory mkdir -p mongodb cp -R -n mongodb-osx-x86_64-3.0.4/ mongodb
  • Ensure the location of the binaries is in the PATH variable export PATH=(mongodb-install-directory)/bin:$PATH
  • Create Default DB Path - "/data/db"
