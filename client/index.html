Introduction

What is MongoDB?

  • MongoDB (from humongous) is a cross-platform document-oriented database
  • Relational database stores data in tables, MongoDB stores data as JSON documents

Why MongoDB?

  • Performance and Scalability
  • Schemaless
  • No SQL
