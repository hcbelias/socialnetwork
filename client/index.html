Introduction

What is MongoDB?

  • First developed by the software company 10gen (now MongoDB Inc.) in October 2007, shifted to an open source development model in 2009
  • Written in C++
  • MongoDB (from humongous) is a cross-platform document-oriented database
  • Relational database stores data in tables, MongoDB stores data as JSON documents

Why MongoDB?

  • Performance and Scalability
  • Schemaless
  • No SQL

Concepts

  • Mongo shell is an interactive Javascript interpreter - you can execute JS fragments!
  • Uses BSON - Binary Javascript Object Notation. BSON extends the JSON model to provide additional data types and to be efficient for encoding and decoding within different languages.

  • Create - Insert / Read - Find / Update - Update / Delete - Remove
